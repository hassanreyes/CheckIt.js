Get Started with CheckIt

Welcome

Hello! And welcome to the CheckIt tool.

Checklists are so useful, but create one is not. Sometimes is hard to know how to start; well, don´t worry we have the solution. CheckIt is a communal compilation of checklists where users can find and create checklists for their need and at the same time help others creating a knowledge base.

For the moment users might only create Software Development related checklists.

 

The following image shows the application layout.

1.       Navigation Bar. A convenient way to go around the application, but also let you do a quick search in the checklist compendium.

2.       Working On area. Registered users can have a “working on” checklist at a time, this area shows an outline of the current authoring checklist.

3.       Navigation Page. The content of this area depends on what are you doing: browsing over checklists, editing your “working on” checklist, etc.

Everyone can browse the compendium of checklists, but only registered users can author.
